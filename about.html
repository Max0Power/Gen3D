3D-Gen

3D-Gen consists of three tools to create 3D models and texture images: world map, noise and user image. You can freely use the 3D models and texture images you have created with 3D-Gen. You can use these, for example, in making maps for games.

This project is a part of our study program in University of Jyväskylä. The main idea of this course is to make a project with a group using open data.

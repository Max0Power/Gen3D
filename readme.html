Getting Started

Prerequisites

  1. Git
  2. Node

Prefer prebuilt binaries over installers.

Installation

  1. Clone repository git clone git@yousource.it.jyu.fi:worldmap3d/worldmap3d.git.
  2. Change directory cd worldmap3d.
  3. Run npm install to install all dependencies.

Usage

  1. Run npm start to start site locally.
  2. Go to localhost:8080 with your browser to open the site.
